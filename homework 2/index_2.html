Какими способами можно обозначить цвет?

  1. Color Names: Tomato, Orange, DodgerBlue, MediumSeaGreen, Gray, Violet ...
  2. RGB values (RED, GREEN, BLUE): rgb(255, 99, 71)
  3. RGBA values (RED, GREEN, BLUE, ALPHA): rgba(255, 99, 71, 0.5)
  4. HEX values (#RRGGBB, where the RR (red), GG (green) and BB (blue)): #ff6347
  5. HSL values (hue, saturation, lightness): hsl(9, 100%, 64%)
  6. HSLA values (hue, saturation, lightness, alpha): hsla(9, 100%, 64%, 0.5)
